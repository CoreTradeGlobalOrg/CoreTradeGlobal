**src/core/validation/changePasswordSchema.js:**
```js
import { z } from 'zod';

export const changePasswordSchema = z.object({
  currentPassword: z.string().min(1, 'Current password is required'),
  newPassword: z.string()
    .min(8, 'Password must be at least 8 characters')
    .regex(/[A-Z]/, 'Must contain at least one uppercase letter')
    .regex(/[a-z]/, 'Must contain at least one lowercase letter')
    .regex(/[0-9]/, 'Must contain at least one number')
    .regex(/[^A-Za-z0-9]/, 'Must contain at least one special character'),
  confirmPassword: z.string().min(1, 'Please confirm your password'),
}).refine((data) => data.newPassword === data.confirmPassword, {
  message: 'Passwords do not match',
  path: ['confirmPassword'],
});
```

**src/core/validation/totpVerifySchema.js:**
```js
import { z } from 'zod';

export const totpVerifySchema = z.object({
  code: z.string()
    .length(6, 'Code must be exactly 6 digits')
    .regex(/^\d{6}$/, 'Code must contain only digits'),
});
```

**src/presentation/hooks/settings/usePasswordChange.js:**
- Export `usePasswordChange()` hook
- Import `auth` from firebase.config, `reauthenticateWithCredential`, `EmailAuthProvider`, `updatePassword` from firebase/auth
- State: `loading`, `error`
- `changePassword(currentPassword, newPassword)` async function:
  1. Set loading=true, error=null
  2. Create credential with `EmailAuthProvider.credential(auth.currentUser.email, currentPassword)`
  3. `await reauthenticateWithCredential(auth.currentUser, credential)` â€” catch auth/wrong-password and set friendly error
  4. `await updatePassword(auth.currentUser, newPassword)`
  5. Return success, set loading=false
  6. Catch errors: map auth/wrong-password -> "Current password is incorrect", auth/requires-recent-login -> "Please log in again", default -> error.message
- Return `{ changePassword, loading, error }`

**src/presentation/hooks/settings/useTwoFactor.js:**
- Export `useTwoFactor()` hook
- Import `multiFactor`, `TotpMultiFactorGenerator`, `reauthenticateWithCredential`, `EmailAuthProvider` from firebase/auth
- Import `auth` from firebase.config
- Import `doc`, `setDoc`, `getDoc` from firebase/firestore and `db` from firebase.config
- State: `step` (idle | reauthenticating | scanning | verifying | showCodes | disabling), `totpSecret` (ref), `qrCodeUrl`, `backupCodes`, `loading`, `error`, `isEnrolled`
- On mount: check `multiFactor(auth.currentUser).enrolledFactors` for totp factorId to set `isEnrolled`
- `startEnrollment()`:
  1. Set step='reauthenticating' â€” caller must provide current password
- `reauthAndGenerateSecret(currentPassword)`:
  1. Reauthenticate with credential
  2. Get multiFactorSession
  3. Generate totpSecret via `TotpMultiFactorGenerator.generateSecret(session)`
  4. Set qrCodeUrl via `totpSecret.generateQrCodeUrl(auth.currentUser.email, 'CoreTradeGlobal')`
  5. Set step='scanning'
- `verifyAndEnroll(sixDigitCode)`:
  1. Create assertion via `TotpMultiFactorGenerator.assertionForEnrollment(totpSecret, sixDigitCode)`
  2. `await multiFactor(auth.currentUser).enroll(assertion, 'TOTP Authenticator')`
  3. Generate 10 backup codes using `window.crypto.getRandomValues(new Uint8Array(5))` per code, format as XXXXX-XXXXX uppercase hex
  4. Hash each code with SHA-256 (use SubtleCrypto: `crypto.subtle.digest('SHA-256', ...)`)
  5. Store hashed codes in Firestore: `setDoc(doc(db, 'users', auth.currentUser.uid, 'security', 'backupCodes'), { codes: hashedCodes, createdAt: new Date() })`
  6. Set backupCodes (plaintext for display), step='showCodes', isEnrolled=true
- `disableTwoFactor(currentPassword)`:
  1. Reauthenticate with credential
  2. Get enrolled factors, find totp factor uid
  3. `await multiFactor(auth.currentUser).unenroll(factorUid)`
  4. Catch `auth/user-token-expired` â€” set error message "You've been signed out. Please log in again." and redirect to /login
  5. Delete backup codes doc from Firestore
  6. Set isEnrolled=false, step='idle'
- `reset()`: Reset all state to idle
- Return `{ step, qrCodeUrl, backupCodes, loading, error, isEnrolled, startEnrollment, reauthAndGenerateSecret, verifyAndEnroll, disableTwoFactor, reset }`
