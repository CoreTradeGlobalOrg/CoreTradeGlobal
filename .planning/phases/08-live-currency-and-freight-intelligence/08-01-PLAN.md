**currencyConstants.js:**
Create a new constants file (do NOT modify existing currencies.js which is for product forms). Define:
- `TICKER_CURRENCIES` array of 8 objects: `{ code, flag, name }` for USD, EUR, GBP, TRY, CNY, JPY, AED, SAR
- `TICKER_PAIRS` array of conventional market quote pairs shown in the marquee. Use both directions for major pairs (e.g., EUR/USD and USD/EUR). Each pair: `{ base, quote, label }` where label is "EUR/USD" format. Include ~12-16 pairs covering all 8 currencies in their conventional quoting direction.
- `FRANKFURTER_BASE_URL = 'https://api.frankfurter.dev/v2'`
- `CURRENCY_CACHE_KEY = 'ctg_live_rates_v1'`
- `CURRENCY_CACHE_TTL = 24 * 60 * 60 * 1000` (24h)
- `POLL_INTERVAL = 60_000` (60s)
- `TARGET_CURRENCY_KEYS = { target1: 'ctg_currency_target_1', target2: 'ctg_currency_target_2' }` (for Plan 02)

**useLiveCurrency.js:**
Create a module-level singleton hook following the pattern from RESEARCH.md Pattern 1. Key requirements:
- Module-level variables: `_rates`, `_fetchedAt`, `_intervalId`, `_subscribers` (Set), `_previousRates` (for arrow indicators)
- `fetchAndNotify()`: calls Frankfurter v2 `GET /rates?base=EUR&quotes=USD,GBP,TRY,CNY,JPY,AED,SAR` — use EUR as base since it is ECB's native base. On success: save `_previousRates = _rates`, update `_rates`, `_fetchedAt = Date.now()`, write to localStorage cache, notify all subscribers. On failure: try loading from localStorage cache; if cache expired (>24h TTL), set error state.
- `document.hidden` check before fetch (skip poll on background tabs per Pitfall 3)
- All localStorage access inside effects or guarded with `typeof window !== 'undefined'` (Pitfall 4)
- Hook returns `{ rates, previousRates, fetchedAt, isStale, cacheExpired, error, loading }` where:
  - `rates` is an object like `{ USD: 1.08, GBP: 0.85, ... }` keyed by currency code, values relative to EUR base
  - `previousRates` is the prior fetch's rates (for arrow direction comparison)
  - `isStale` = true if `fetchedAt` is from cache and > 60s old
  - `cacheExpired` = true if > 24h with no successful fetch
- First subscriber starts interval, last subscriber clears it
- Helper function exported: `convertAmount(amount, fromCurrency, toCurrency, rates)` — converts any amount between any two of the 8 currencies using EUR as intermediary base. Returns null if rates unavailable.

**CurrencyTicker.jsx:**
Create a `'use client'` component that renders an auto-scrolling marquee bar. Requirements:
- Calls `useLiveCurrency()` to get `{ rates, previousRates, fetchedAt, isStale, cacheExpired, error, loading }`
- Layout: `overflow-hidden` wrapper with `bg-[#0A1628]` (slightly darker than main bg), `border-b border-[#2A3B52]`, `py-1.5`
- Inside: a flex container with `animate-marquee whitespace-nowrap` class. Content is DUPLICATED (two identical `<TickerItems />` children) to create a seamless infinite loop.
- Each ticker item renders a pair from `TICKER_PAIRS`: gold `#FFD700` text for the pair label (e.g., "EUR/USD"), white text for the rate value. Calculate rate from `rates` object: if pair base is EUR, use `rates[quote]` directly; if pair base is not EUR, compute as `rates[base] ? 1/rates[base] * rates[quote]` or use the `convertAmount` helper.
- Arrow indicators: compare current rate vs `previousRates` for the same pair. Show green `▲` if rate increased, red `▼` if decreased, gray `–` if unchanged or no previous data. Use `text-emerald-400` for up, `text-red-400` for down.
- Items separated by a subtle dot divider `·` in `text-[#4A5568]`
- At the end of each ticker content block, append "Updated Xs ago" using `formatDistanceToNow` from date-fns on `fetchedAt`. Style: `text-[#8899AA]` italic.
- Loading state: show `animate-pulse` skeleton bar matching the ticker height
- Error/cacheExpired state: show "Currency rates temporarily unavailable" in muted text, no marquee animation
- Stale state (isStale=true): show the rates normally but append a small warning badge "(rates may be delayed)" in `text-amber-400`
- Responsive: base text `text-[10px]` on mobile, `sm:text-xs` on desktop per user decision (same marquee, smaller text on mobile)
- Pause marquee on hover: add `hover:[animation-play-state:paused]` to the animate-marquee container for usability
