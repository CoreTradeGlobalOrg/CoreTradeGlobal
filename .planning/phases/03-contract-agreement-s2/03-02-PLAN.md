**2. Create `src/presentation/hooks/contract/useContractActions.js`:**

`'use client'` directive. Takes `(dealId, contract, currentUid, deal)` as parameters.

Manages local state:
- `localApprovedClauses` — `useState(new Set())` initialized from `contract.getMyApproval(currentUid, deal).approvedClauses` on mount and when contract changes (only if not yet submitted)
- `expandedSections` — `useState(new Set())` for currently expanded accordion sections
- `hasExpanded` — `useState(new Set())` tracking which sections have ever been expanded (persists across collapses)
- `loading` — `useState(false)` for submit button

Initialize `hasExpanded` from contract data on load: any section containing a clause that appears in `approvedClauses` should be treated as already expanded (restores state after refresh — per Pitfall 6 in research).

Functions:
- `toggleSection(sectionId)` — toggles expandedSections Set; adds to hasExpanded Set
- `toggleClause(clauseId)` — toggles in localApprovedClauses Set; triggers debounced save
- `saveDraft(approvedClauses)` — debounced (500ms) call to `httpsCallable(functions, 'saveDraftApprovals')({ dealId, approvedClauses: Array.from(approvedClauses) })`. Wrap in try/catch, log errors but don't show toast (draft saves are silent).
- `submitApprovals()` — sets loading=true, calls `httpsCallable(functions, 'submitContractApproval')({ dealId })`. On success: `toast.success('Approvals submitted')`. On error: `toast.error(error.message)`. Sets loading=false.
- `isCheckboxActive(clause)` — returns `hasExpanded.has(clause.section) && !myApproval.hasSubmitted`
- `isClauseApproved(clauseId)` — checks localApprovedClauses Set

Implement debounce inline with `useRef` + `setTimeout`/`clearTimeout` (no lodash needed):
```javascript
const saveTimerRef = useRef(null);
function debouncedSave(clauses) {
  if (saveTimerRef.current) clearTimeout(saveTimerRef.current);
  saveTimerRef.current = setTimeout(() => saveDraft(clauses), 500);
}
```

Return: `{ localApprovedClauses, expandedSections, hasExpanded, loading, toggleSection, toggleClause, submitApprovals, isCheckboxActive, isClauseApproved }`
