**1. `src/presentation/hooks/quote/useQuotesForDeal.js`** — Real-time subscription for buyer quotes view:
- 'use client' directive
- Takes dealId as parameter
- Uses two-level subscription:
  1. Subscribe to quoteRequests for deal via container.getQuoteRequestRepository().subscribeToRequestsForDeal(dealId, ...)
  2. For each quoteRequest, subscribe to its providerQuotes via container.getQuoteRepository().subscribeToQuotesForRequest(requestId, ...)
- State: { quoteRequests: [], allQuotes: [], loading: true }
- Derived (useMemo):
  - `insuranceQuotes`: allQuotes filtered where providerType === 'insurance' AND status === 'active' (not withdrawn/expired)
  - `logisticsQuotes`: allQuotes filtered where providerType === 'logistics' AND status === 'active'
  - `selectedInsuranceQuote`: find quote with status 'accepted' and providerType 'insurance'
  - `selectedLogisticsQuote`: find quote with status 'accepted' and providerType 'logistics'
  - `insuranceRequestCount`: count of insurance quoteRequests
  - `logisticsRequestCount`: count of logistics quoteRequests
  - `declinedCount`: count of declined quoteRequests
- Cleanup: unsubscribe all on unmount (track in useRef array)
- Returns { quoteRequests, insuranceQuotes, logisticsQuotes, selectedInsuranceQuote, selectedLogisticsQuote, insuranceRequestCount, logisticsRequestCount, declinedCount, allQuotes, loading }

- Render QuotesPage with: deal, insuranceQuotes, logisticsQuotes, selectedInsuranceQuote, selectedLogisticsQuote, quoteRequests, currentUserUid, actions (from useQuoteActions), isBuyer (currentUid === deal.buyerId)
