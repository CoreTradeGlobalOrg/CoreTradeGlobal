       - Returns: `{ deal, contract, selectedInsuranceQuote, selectedLogisticsQuote, shipmentUpdates, latestShipment, legalEngagement, loading, error }`
       - Cleanup: return all unsubscribe functions in useEffect cleanup

       - Props: `{ deal, latestShipment }`

       - Responsive: stacks on mobile, horizontal on desktop

    3. **TradeInfoBar.jsx** — Horizontal info strip below hero:
       - Props: `{ deal, latestShipment }`
       - Shows: Deal No (deal.id truncated), Product, Total Amount, Incoterms, Container # (from latestShipment.containerNumber or 'Pending'), Status badge
       - Dark theme: bg-[#0b1626] border-[#1e2d47], evenly spaced items

    4. **DealOverviewSection.jsx** — Product, price, Incoterms, quantities:
       - Props: `{ deal }`

       - Styled as a card section with header and key-value pairs

    5. **PartiesProvidersSection.jsx** — Buyer/seller + selected providers:
       - Props: `{ deal, selectedInsuranceQuote, selectedLogisticsQuote }`
       - Shows buyer and seller info (name, company)
       - Shows selected insurance provider (if any): company name, ICC coverage type, premium, validity
       - Shows selected logistics provider (if any): company name, transport mode, cost, timeline

    6. **CostBreakdownSection.jsx** — Product + insurance + logistics = total:
       - Props: `{ deal, selectedInsuranceQuote, selectedLogisticsQuote }`
       - Product cost from deal.latestOfferSnapshot.price * quantity
       - Insurance premium from selected insurance quote (or 'Pending')
       - Logistics fee from selected logistics quote (or 'Pending')
       - Total = sum of available values
       - Visual: table-like layout with separator line and bold total

    7. **DocumentsSection.jsx** — Contract PDF + placeholder slots:
       - Props: `{ deal, contract }`
       - Shows contract PDF download link if contract exists (link to /deals/{dealId}/contract)
       - Placeholder slots for future documents: Insurance Certificate, Bill of Lading, Commercial Invoice, Packing List — each showing 'Not yet available' with a document icon
       - These are informational placeholders, not upload functionality

       - If no engagement: show 'No legal counsel engaged for this deal'
       - If engagement exists: show lawyer name, engagement status (active/completed), number of drafts, latest draft approval status
       - CRITICAL: Only display the engagement where `engagement.clientId === currentUserUid`. Never show opposing party's lawyer info.

    9. **TradeRouteMap.jsx** — Static SVG map with dark theme:

       - Use static placeholder positions (left side for origin, right side for destination) — exact coordinates not required per user decision ("purely visual context")
