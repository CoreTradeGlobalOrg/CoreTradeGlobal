1. Create `src/presentation/hooks/legal/useLegalEngagement.js`:
   - Subscribes to a single engagement for a specific deal + client (for LegalBanner on DealPage)
   - Parameters: `(dealId, clientId)` -- both required, returns null if either is falsy
   - Uses `container.getLegalEngagementRepository().subscribeToEngagementForDeal(dealId, clientId, callback)`
   - Returns: `{ engagement, loading, error }`
   - Cleanup: unsubscribe on unmount (standard useEffect cleanup pattern)
   - Follow existing hook patterns (useContract, useQuotesForDeal)

2. Create `src/presentation/hooks/legal/useLegalEngagements.js`:
   - Subscribes to ALL engagements for a lawyer (for LawyerDashboard)
   - Parameters: `(lawyerId)` -- required, returns empty if falsy
   - Uses `container.getLegalEngagementRepository().subscribeToEngagementsForLawyer(lawyerId, callback)`
   - Returns: `{ engagements, loading, error }`
   - Derived state (useMemo):
     - `pendingEngagements`: filter where status === 'pending'
     - `activeEngagements`: filter where status === 'active'
     - `completedEngagements`: filter where status === 'completed'
   - Return all four: `{ engagements, pendingEngagements, activeEngagements, completedEngagements, loading, error }`

3. Create `src/presentation/hooks/legal/useLegalActions.js`:
   - Provides hire, respond, and close actions using httpsCallable
   - Import: getFunctions, httpsCallable from 'firebase/functions'
   - State: `loading` (bool), `error` (string|null)
   - Methods:
     a) `hireLawyer(dealId, lawyerId)` -- calls `httpsCallable(functions, 'hireLayyer')({ dealId, lawyerId })`. Wrap in try/catch, set loading, handle errors with toast. Returns result.data.
     b) `respondToHireRequest(engagementId, action)` -- calls `httpsCallable(functions, 'respondToHireRequest')({ engagementId, action })`. action is 'accept' or 'decline'. Returns result.data.
     c) `closeLegalEngagement(engagementId)` -- calls `httpsCallable(functions, 'closeLegalEngagement')({ engagementId })`. Returns result.data.
     d) `submitReview(engagementId, rating, comment)` -- calls `httpsCallable(functions, 'submitLawyerReview')({ engagementId, rating, comment })`. Returns result.data.
   - Returns: `{ hireLawyer, respondToHireRequest, closeLegalEngagement, submitReview, loading, error }`
   - Error handling: catch error, extract error.message from HttpsError, set error state, show toast.error()
